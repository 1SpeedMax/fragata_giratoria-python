📋 GUÍA DE CONFIGURACIÓN DEL PROYECTO
LA FRAGATA GIRATORIA (DJANGO + POSTGRESQL)
1. Introducción

Este documento describe el proceso completo para la instalación, configuración y ejecución del proyecto web desarrollado en Django denominado “La Fragata Giratoria”.
Incluye la configuración del entorno, base de datos, modelos y solución de errores comunes, permitiendo que cualquier desarrollador pueda levantar el sistema correctamente.

2. Prerrequisitos

Antes de iniciar, se debe contar con los siguientes programas instalados:

Python 3.12
PostgreSQL
pgAdmin (opcional, recomendado)

3. Configuración del Entorno Virtual

Para aislar las dependencias del proyecto:

3.1 Crear el entorno virtual
python -m venv venv
3.2 Activar el entorno

Windows

Instalar las librerías necesarias:

17. Solución de Errores Comunes
Error: tabla auth_user no existe
python manage.py migrate
Error: columna fecha_creacion no existe
ALTER TABLE usuarios_usuario ADD COLUMN fecha_creacion TIMESTAMP DEFAULT NOW();
Error: columna nombre_usuario
ALTER TABLE usuarios_usuario ADD COLUMN nombre_usuario VARCHAR(150);
UPDATE usuarios_usuario SET nombre_usuario = username;
Error: columna nombre_metodo
ALTER TABLE metodos_pago RENAME COLUMN nombre TO nombre_metodo;
Error: tabla metodos_pago no existe

Crear la tabla manualmente en PostgreSQL.

from django.db import migrations

def create_initial_data(apps, schema_editor):
    Rol = apps.get_model('usuarios', 'Rol')
    Rol.objects.create(nombre_rol='ADMIN', descripcion='Administrador')

class Migration(migrations.Migration):
    dependencies = [('usuarios', '0001_initial')]
    operations = [migrations.RunPython(create_initial_data)]

# La Fragata Giratoria

## Requisitos
- Python 3.12
- PostgreSQL

## Instalación
1. Clonar repositorio
2. Crear entorno virtual: `python -m venv venv`
3. Activar: `venv\Scripts\activate`
4. Instalar: `pip install -r requirements.txt`
5. Crear archivo `.env` (ver `.env.example`)
6. Crear base de datos: `CREATE DATABASE fragata_db1;`
7. Migrar: `python manage.py migrate`
8. Cargar datos: `python manage.py loaddata fixtures/*.json`
9. Crear superusuario: `python manage.py createsuperuser`
10. Ejecutar: `python manage.py runserver`
